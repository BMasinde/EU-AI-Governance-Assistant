# Ideas and Project Trajectory

With many of the GDPR and EU AI Act laws comming into place, it is becoming more 
important to be aware of these regulations and how to comply with them; especially 
if one work's with applications at the intersection of these regulations. 

In this project I want to build an RAG model that I can query the various
regulations. I would like to build it from scratch using as little frameworks as 
possible and relying on mostly opensource platforms. I would also like an agentic 
layer (idea still being developed) which can take the following actions: 

    * Analayze obligations for various persons in an organization.
    * Compare regulatory requirements for a particular use case; across the 
    various regulations.
    * Trace definitions or how terms are used among the various regulations.
